Welcome to the Labeling System

The Labeling System project develops a prototype of a web based editor for controlled vocabularies. The system relies on the concept of linked-data implementing technologies and formats to semantically model the relationship and meaning of concepts from a set of thesauri. The resulting RDF graphs representing the vocabularies are accessible via a REST interface and suitable for fostering research across scientific disciplines.
